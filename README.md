這是一臺使用NodeMCU ESP8266開發板作爲核心的遙控戰鬥機器人，使用Blynk作爲其控制端介面程式，只需將機器人連上Wi-Fi，即可進行遠端操控。

## 開始製作

請跟隨以下順序進行操作

- 創造自己的機器人 [CAD](cad/README.md)
- 控制端設置 [Blynk配置](blynk/README.md)
- 寫入程式 [Arduino Code](firmware/README.md)
